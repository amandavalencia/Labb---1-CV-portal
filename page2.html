LIA4U

Detta projekt utvecklades som ett skolprojekt tillsammans med mina klasskamrater. Målet var att skapa ett verktyg där studenter kan söka på ett yrke inför sin LIA och få upp relevanta företag som potentiellt erbjuder praktikplatser.

Applikationen hämtar data från Arbetsförmedlingens öppna API för jobbannonser och presenterar resultaten i ett enkelt och sökbart gränssnitt. Designen är inspirerad av Arbetsförmedlingens designsystem för typografi och struktur, men med ett eget färgtema.

Några saker jag fokuserade på:

  • Dynamisk hämtning och rendering av data från ett externt API
  • Strukturerad CSS med custom properties för ett mer konsekvent designsystem
